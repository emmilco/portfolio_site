Portfolio

Some of my recent projects

Howl Live Link

A full-stack, single page application inspired by Medium

I built Howl in React/Redux with a Rails backend, PostgreSQL database, and AWS S3 image hosting. The most exciting feature of the app is its pure WYSIWYG article editor, which I built from scratch using HTML contenteditable and modular content modeling to allow for the dynamic creation and deletion of HTML elements, including support for embedded images and YouTube videos. Click the link above to try it out; it's really cool!

PathFinder Live Link

An interactive exploration of Maze-related algorithms

Pathfinder was built in JavaScript ES6 using HTML5 Canvas. It takes users on a visual tour of the ways different algorithms and datastructures can be used to generate mazes and maze solvers. Users are introduced to a variety of algorithms and shown some of their interesting structural properties. It includes sections on Prim's algorithm, Dijkstra's algorithm, A*, and others.

Ruborm View Project

A lightweight Object-Relational Mapping built in Ruby for SQLite3

I built Ruborm as an exercise to understand how ORMs work on the inside. It includes all the basic CRUD functionality you would expect from an ORM, plus convenience methods, custom associations, and support for multi-layered associations.
